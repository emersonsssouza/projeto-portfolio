Prefeitura de Igaratá

Este é o código-fonte de um site institucional para a Prefeitura de Igaratá. Ele foi projetado para fornecer informações importantes sobre o município e seus serviços aos cidadãos.

Sistema solar

Este é um projeto de página da web que apresenta o Sistema Solar de forma visual e auditiva. Ele é composto principalmente por um arquivo HTML, mas também faz referência a um arquivo CSS para estilização e a um arquivo de áudio para uma trilha sonora relacionada ao espaço.

Formulario

Este projeto consiste em uma página da web simples que exibe um formulário de login. É uma página HTML básica que utiliza algumas tags e estilos CSS para criar uma interface de login.

Lampada JS

Este é um projeto simples de página da web que simula o controle de uma lâmpada usando JavaScript. A página possui uma interface básica com botões para ligar, desligar e quebrar a lâmpada.

Android

Este projeto é uma página da web que oferece informações interessantes sobre o mascote do sistema operacional Android, bem como curiosidades relacionadas a esse sistema operacional.

RDK Molas

Este projeto consiste em uma página web que representa a RDK Indústria e Comércio de Molas. A página possui informações sobre a empresa, seus produtos, máquinas, certificações e informações de contato.
